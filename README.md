# インベーダーゲーム

これは、HTML、CSS、およびJavaScriptで作成されたシンプルなインベーダースタイルのゲームです。

## 遊び方

- **左矢印キー**: プレイヤーを左に移動します
- **右矢印キー**: プレイヤーを右に移動します
- **スペースバー**: 弾丸を発射します

## セットアップ

1. このリポジトリをクローンします:
   ```bash
   git clone https://github.com/(Your-GitHub-Username)/invaders-game-2.git
   ```
2. `index.html` ファイルをウェブブラウザで開きます。
